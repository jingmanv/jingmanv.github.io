MacOS Sierra使用PPTP协议连接VPN

Jun 22, 2017 | mac | 阅读

截止目前，最新版的MacOS Sierra已经取消PPTP协议连接VPN，网上找到替代方案，经本人实际配置证实可用，配置如下：

  1. 系统偏好设置 -> 网络 -> “+” 。
  2. 接⼝选择”VPN”，VPN类型选择”IPSec上的L2TP”，服务名称随意。
  3. 对刚创建的VPN进⾏行行配置，根据实际vpn地址填写服务地址、账户名称。
  4. 鉴定设置… -> 用户鉴定中，勾选密码，填写vpn密码，保存关闭。
